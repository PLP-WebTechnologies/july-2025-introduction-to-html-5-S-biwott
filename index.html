Understanding Semantic HTML

Semantic tags give meaning to your HTML content. Instead of using div everywhere, HTML5 provides tags like:

  • header – Introduces the page or a section
  • main – Primary content of the document
  • section – Groups related content together
  • article – Independent, self-contained content
  • footer – Closing or supplemental info

Accessibility Features

Good websites are usable by everyone. This includes people using screen readers or keyboard navigation. To improve accessibility:

  • Provide descriptive alt text for images
  • Use aria-label to describe navigation or landmarks
  • Maintain heading order without skipping levels
